This plugin adds necessary code to make Wordpress SEO by Yoast compatible with BBPress.

== Description ==
Yoast SEO BBpress Addon makes Wordpress SEO by Yoast compatible with BBPress by adding simple codes. The plugin requires Wordpress SEO by Yoast to work. When installed successfully, the topics in BBpress displays the correct meta description from Topic Content. This is a simple plugin and do not alter any codes in Wordpress SEO plugin neither in Wordpress. 
The plugin  has originally been coded for  a Health site and working live with BBpress as its forum in (https://ehospital.in/)
Any support questions can be asked in the support forums of eHospital. 

== Installation ==
Upload the entire yoast-seo-bbpress-addon folder to the /wp-content/plugins/ directory
DO NOT change the name of the yoast-seo-bbpress-addon folder
Activate the plugin through the ‘Plugins’ menu in WordPress
